# AppCleaner Mac 🚀

**AppCleaner Mac** is a premium, minimalist utility designed to help you uninstall applications and find all their related hidden files (caches, logs, preferences) with a single click.

---

## 🌟 Key Features
- **Deep Scan:** Locates hidden files buried deep in your system library.
- **Full Disk Access Support:** Ensures no cache file is left behind.
- **Minimalist Design:** A clean, logo-free interface following modern macOS design language.
- **Trash-First Safety:** Files are moved to Trash, giving you a chance to review before permanent deletion.
- **Performance Optimized:** Fast scanning with zero background overhead.

---

## 🛠 Installation & Usage

1. **Download:** Get the latest `AppCleanerMac.zip` from the [Releases](https://github.com/brandskitstudio/AppCleanerMac/releases) section.
2. **Setup:** Unzip and drag the application into your **Applications** folder.
3. **Permissions:** On the first launch, the app will guide you to enable **Full Disk Access**. This is required to find and delete hidden system files.
4. **Clean:** Simply drag any app into the window or use the "Choose App" button to start deep cleaning.

---

## 🔒 Security & Privacy
This app values your privacy. 
- **Zero Tracking:** No data collection or telemetry.
- **Offline First:** No server connections or internet access required.
- **Local Processing:** All scanning and deletion happen entirely on your machine.
